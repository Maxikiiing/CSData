Kurzfassung

  • Die App verarbeitet keine personenbezogenen Daten des Nutzers.
  • Leaderboards werden nur lokal auf dem Gerät (UserDefaults) gespeichert.
  • Bingo-Boards und Spielerdaten werden per HTTPS von GitHub Pages geladen; dabei werden keine Nutzerkennungen übertragen.
  • Es gibt derzeit keine Werbung und keine Analytics/Tracking-Dienste.

Nutzungsdaten & Speicherorte

  • Lokale Daten: Spielstände/Leaderboards werden lokal auf deinem Gerät gespeichert und nicht synchronisiert.
  • Netzwerkzugriffe: Die App lädt statische JSON-Dateien (Bingo-Boards, Spielerdaten) von github.io über HTTPS. Es werden keine Cookies gesetzt und keine personenbezogenen Daten übertragen.

Geräteberechtigungen

Die App nutzt keine Kamera, Kontakte, Fotos, Ortungsdienste o. Ä. Es werden lediglich Haptik/Vibrationen des Geräts für Feedback verwendet.

Drittanbieter

  • GitHub Pages (Bereitstellung statischer Inhalte). GitHub kann aus technischen Gründen Server-Logdaten verarbeiten (z. B. bei Abruf der Seiten). Diese Daten sind uns nicht personenbezogen zuordenbar.
  • Optional in Zukunft: Analytics/Werbung. Sollten solche Dienste eingeführt werden, wird diese Erklärung vorab aktualisiert.

Rechtsgrundlagen

Da keine personenbezogenen Daten verarbeitet werden, finden DSGVO-Rechtsgrundlagen auf die App selbst derzeit keine Anwendung. Beim Aufruf der über GitHub Pages gehosteten Seiten gelten die dortigen technischen Notwendigkeiten (HTTPS-Abruf).

Ihre Rechte

Sofern künftig personenbezogene Daten verarbeitet würden, hätten Sie insbesondere das Recht auf Auskunft, Berichtigung, Löschung, Einschränkung der Verarbeitung, Widerspruch sowie Datenübertragbarkeit. Aktuell werden von der App keine personenbezogenen Daten verarbeitet.

Datensicherheit

Alle externen Inhalte werden ausschließlich verschlüsselt (HTTPS) geladen. Die App betreibt keinen eigenen Server und speichert keine personenbezogenen Daten.

Änderungen

Diese Erklärung kann angepasst werden. Wesentliche Änderungen werden in der App oder an dieser Stelle bekanntgegeben. Bitte prüfen Sie das Datum „Zuletzt aktualisiert“.
